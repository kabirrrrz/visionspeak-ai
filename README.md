# VisionSpeak AI

VisionSpeak AI is a full-stack AI web application that converts images and text into speech.

## Features

- OCR image to text using Tesseract
- Editable extracted text
- Text-to-speech conversion
- Multi-language speech generation
- Drag & drop image upload
- Dark mode UI
- Audio playback and download
- Modern React interface

## Tech Stack

Frontend
- React
- Tailwind CSS
- Axios

Backend
- Flask
- Tesseract OCR
- gTTS

## How It Works

Upload Image → OCR → Edit Text → Convert to Speech → Play / Download Audio
